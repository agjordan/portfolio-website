About Me

I’m a London-based software engineer. Up until recently, I was a Product Manager who really enjoyed working with teams to create amazing products. I felt a calling to be more hands-on in the creation of products and have since explored more of the coding side of product development.

I’m equally interested in the front and back end where I find constant challenges, but that is also where I like to live, just on the edge of comfort where I am still learning. I feel most confident coding in Javascript, TypeScript, and Python but I have dabbled in other languages such as C, Lua, and Ruby.

Portfolio

Just my Recipes

Built a TypeScript and React based Single Page App to save recipes from the internet without any of the 'fluff' commonly found around recipes (the author's life story, how they came up with the recipe, etc.).

The App is supported by a simple Chrome Extension that makes it easier to save recipes (but a URL can also be pasted in the app to save). It is hosted on Firebase with a simple API endpoint on Heroku.

Work in Progress - next steps are to roll out support for different data formats. Currently it works very well on more recent articles hosted on Food52.com (which happens to also be my favourite food blog/website), your mileage will vary with other sources.

Pen and Paper Games

Morse Code

Morse code -> English -> Morse translator built in vanilla JS. All translation capabilities written by me without libraries. Implemented basic audio output with custom sounds (created in bxfr) for Morse code and used Audio and SpeechSynthesisUtterance to implement Text to Speech in English.

GitHub | Deployed site
